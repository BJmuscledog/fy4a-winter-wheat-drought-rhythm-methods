## Suggested title

FY-4A winter wheat drought-rhythm methods (v1.0.0)

## Suggested description

This Figshare record archives version `v1.0.0` of the companion-code repository
for the *Remote Sensing of Environment* study on resolving daytime
photosynthetic rhythms of winter wheat under drought stress from FY-4A
observations.

The archive includes the key method code, configuration, and documentation used
to reproduce the manuscript workflow, assuming access to the original FY-4A,
VIIRS, TROPOSIF, and meteorological source datasets.

Raw source datasets are not redistributed in this archive because access is
governed by the original providers and their respective access policies.
